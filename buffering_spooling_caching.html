La Paradoja de la Velocidad

Los procesadores son órdenes de magnitud más rápidos que los dispositivos de almacenamiento.

~1 ns Registros CPU
~100 ns RAM Principal
~30 s Impresora (pág)

Brecha de hasta 10 millones de veces.

§1 Buffering

Reserva de memoria RAM para almacenar datos temporalmente durante transferencias.
  • Normaliza velocidades entre productor/consumidor.
  • Ajusta tamaños de bloque.
  • Permite E/S no bloqueante.

Double Buffering

El "estándar de oro" del paralelismo.

CPU ⮕ [ Buffer A ] ⮕ Hardware
       [ Buffer B ] (Llenando...)

Mientras se vacía uno, se llena el otro.

§2 Spooling

(Simultaneous Peripheral Operations On-Line)

Usa el disco como un buffer masivo para interceptar trabajos periféricos.

Convierte un dispositivo de acceso exclusivo en uno virtualmente compartido.

§3 Caching

Mantiene copias de datos lentos en memoria rápida basándose en la Localidad.
Temporal
Se usará de nuevo pronto.
Espacial
Se usará lo que está cerca.

La diferencia crucial

Un Buffer es el único lugar donde reside el dato.

Una Caché es solo una copia redundante.

Si borrás la caché, el sistema es lento pero los datos sobreviven.

Práctica: PrintNightmare

CVE-2021-34527

El problema: RpcAddPrinterDriverEx() permitía a cualquier usuario instalar drivers (DLLs) maliciosos.

¿Por qué SYSTEM? El spooler necesita privilegios para cargar drivers y escribir en System32\spool.

Solución: Privilege Separation.
